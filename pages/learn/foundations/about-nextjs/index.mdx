熟悉 JavaScript、React 及相關的網頁開發概念，能夠有效率的使用 Next.js。可是 Javascript 和 React 是一個廣大的議題，要如何得知自己有能力開始學習 Next.js？

歡迎來到 Next.js 基礎課程！這個課程對於新手非常友好且提供許多範例演練，在這裡我們將會一步一步帶著你了解學習 Next.js 之前所需要的相關知識。首先，利用 Javascript 一步步建立一個簡單的應用程式，完成之後，再把它轉變成 React、Next.js 的形式。

每個區塊的內容將延伸上一個區塊討論的議題，這讓你可以依據你目前的程度決定從哪裡開始。

<Blockquote>
  此教學建立於你已經對於 HTML, CSS, JavaScript 有一定的了解，但沒有 React 相關知識。所以，如果你已經對於 React 相當熟悉，請至
  **[快速開始 Next.js](/learn/foundations/from-react-to-nextjs/getting-started-with-nextjs)**
  或
  **[打造你的第一個 Next.js 應用程式](/learn/basics/create-nextjs-app)**
  。
</Blockquote>

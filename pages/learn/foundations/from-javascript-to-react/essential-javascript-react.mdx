### React 中的基礎 JavaScript

雖然你可以同時學習 JavaScript 和 React，但熟悉 JavaScript 可以使學習 React 的過程更加輕鬆。

在接下來的章節中，我們將從 JavaScript 的角度介紹 React 的一些核心概念。以下是會提到的 JavaScript 主題的摘要：

- [函數](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Functions)和[箭頭函數](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Functions/Arrow_functions)
- [物件](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object)
- [陣列和陣列方法](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array)
- [解構](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Destructuring_assignment)
- [文字模板](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Template_literals)
- [三元運算子](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Conditional_Operator)
- [ES 模塊和導入/導出語法](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Modules)

雖然本課程不深入探討 JavaScript，但了解最新版本的 JavaScript 總是好的。如果你還沒有精通 JavaScript，請不要讓這阻礙你開始使用 React 建構你的應用！

### 如何繼續學習 React

目前為止，我們已經向你介紹完三個基本的 React 概念：**元件**、**屬性** 和 **狀態**。在這些方面擁有扎實的基礎將幫助你開始構建 React 應用程式。一旦你感到更有信心，還可以查看以下其他 React 主題：

- [React 如何處理渲染](https://beta.reactjs.org/learn/render-and-commit) 以及 [如何使用 refs](https://beta.reactjs.org/learn/referencing-values-with-refs)
- [如何管理狀態](https://beta.reactjs.org/learn/managing-state)
- [如何在深度巢狀的資料中使用 React context](https://beta.reactjs.org/learn/passing-data-deeply-with-context)
- [如何使用 React API Hooks](https://beta.reactjs.org/apis/react)，例如 `useEffect()`

### React 學習資源

經過多年積累，已有許多課程、影片和文章來幫助開發者學習 React。雖然很難推薦適合每個人學習風格的資源，但一個非常棒的參考資料是 [React 文檔](https://beta.reactjs.org/learn)，其中包含互動式沙盒來讓你練習 React。

在學習 React 時，**最好的學習方式是動手做**。你可以透過使用 `<script>` 以及到目前為止所學的知識，來使用 React 將小元件添加到現有網站。然而，許多開發者發現，React 的使用者和開發者體驗足以讓他們深入研究 React ，並用它來編寫整個前端專案。

### 從 React 到 Next.js

雖然 React 擅長構建 UI，但要獨立地構建出一個功能齊全的可擴展應用程式確實需要一些開發成本。好消息是，Next.js 幫我們處理好了大部分設置和配置上的工作，並具有幫助你構建 React 應用程式的額外功能。

接下來，我們會將範例從 React 遷移到 Next.js，討論 Next.js 的工作原理，並介紹一些 Web 開發概念，以幫助你學習更進階的 Next.js 功能。
